# WhatsApp Sender

A simple WhatsApp dashboard for bulk messaging and customer management.

## Features

- Add and manage customers
- Create and manage message templates
- Schedule bulk messages
- Open multiple WhatsApp chats with pre-filled messages
- Export/Import customers via CSV
- Quick Send feature for individual messages

## Usage

1. Open the HTML file in a web browser
2. Add customers with their phone numbers
3. Create or select a message template
4. Set the date and time variables
5. Generate messages
6. Open all in WhatsApp or send individually

## Technology

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for data persistence
- WhatsApp Web API
